
# Python Security & Networking Practice Scripts

A collection of small Python scripts focused on cybersecurity, networking, and system utilities.

## Contents

- Password strength checkers (basic + regex)
- Log file analysis
- Open port checker
- Internet connectivity tester
- File type counter
- Password list matcher (wordlist check)
- File integrity checker (MD5)
- Keylogger (keyboard + mouse events)
- Caesar cipher encryption
- Packet sniffing (Scapy)
- Base64 encoding/decoding
- CPU usage monitor
- Password pattern generator
- Security headers checker
- Secure password generator
- Brute force detection logic
- Open redirect checker
- Hash cracking (SHA1 wordlist)
- Disposable email detection
- HaveIBeenPwned password check

## Requirements

Some scripts require external libraries:

```

pip install requests pynput scapy psutil

```

## Usage

Run any script individually:

```

python program01.py

```

## Notes

- These scripts are for educational purposes only.
- Some tools (keylogger, sniffing, etc.) may require permissions.
